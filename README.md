# Neuro-sama 静态网页项目

（这是我帮别人写的静态网页中我最喜欢的一个）

这是一个关于全球顶流 AI VTuber Neuro-sama 的静态网页项目，展示了 Neuro-sama 的详细信息、发展历程、能力特色以及粉丝社群等内容。

## 项目概述

该项目是一个精心设计的静态网页，旨在全面介绍由英国开发者 Vedal 打造的全 AI 驱动虚拟主播 Neuro-sama。网站采用现代化的设计风格，响应式布局，提供了丰富的内容和良好的用户体验。

## 功能特点

### 🎨 现代化设计

- 采用紫色主题，符合 Neuro-sama 的品牌调性
- 响应式布局，适配桌面端和移动端
- 平滑滚动效果和交互动画
- 卡片式设计和渐变背景

### 📱 核心功能

1. **主页导航** - 包含关于 Neuro、Vedal 与开发、发展时间线、多媒体、粉丝社群等导航链接
2. **英雄区** - 展示 Neuro-sama 的核心信息和主要链接
3. **核心简介** - 介绍 Neuro-sama 的基本信息和关键数据
4. **详细信息** - 包括 Vedal 与开发背景、Neuro 的能力与特色、发展时间线、人格设定
5. **多媒体展示** - 展示 Neuro-sama 的视频内容和精彩瞬间
6. **粉丝社群** - 提供各平台的粉丝社区链接
7. **图片查看器** - 点击 Logo 可查看放大版本

### 🔧 技术实现

- **HTML5** - 语义化标签结构
- **CSS3** - 现代样式设计，包括 Flexbox、Grid 布局
- **JavaScript** - 实现交互功能，如汉堡菜单、平滑滚动
- **响应式设计** - 适配不同屏幕尺寸

## 页面结构

### 主页面 (neuro-sama.html)

1. **顶部导航栏** - 固定在顶部，包含 Logo 和导航菜单
2. **英雄区** - 展示 Neuro-sama 的核心信息和主要按钮
3. **核心简介区** - 介绍 Neuro-sama 的基本信息和关键数据
4. **详细信息区** - 包含多个子部分：
   - Vedal 与开发背景
   - Neuro 的能力与特色
   - 发展时间线
   - 人格设定
5. **多媒体展示区** - 展示 Neuro-sama 的视频内容
6. **粉丝社群区** - 提供各平台的粉丝社区链接
7. **页脚** - 包含版权信息、快速导航和免责声明

### 图片查看器 (image-viewer.html)

- 居中展示 Neuro-sama Logo 的放大版本
- 提供返回主页面的链接

## 技术亮点

1. **响应式设计** - 使用媒体查询和弹性布局，确保在不同设备上的良好显示效果
2. **交互体验** - 实现了平滑滚动、悬停效果、汉堡菜单等交互功能
3. **视觉效果** - 使用渐变背景、阴影、卡片设计等现代 UI 元素
4. **性能优化** - 使用 WebP 格式图片，优化加载速度
5. **代码结构** - 清晰的 HTML 结构和模块化的 CSS 代码

## 如何使用

1. 克隆或下载项目到本地
2. 直接打开 `neuro-sama.html` 文件即可查看主页面
3. 点击 Logo 可进入 `image-viewer.html` 查看放大的 Logo
